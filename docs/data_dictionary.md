# IPL360 Data Dictionary

## Matches

| Column | Description |
|----------|-------------|
| match_id | Unique match identifier |
| date | Match date |
| venue | Venue |
| team1 | Team 1 |
| team2 | Team 2 |
| winner | Match winner |
