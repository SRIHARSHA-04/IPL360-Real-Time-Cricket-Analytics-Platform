# IPL360 – Real-Time Cricket Analytics Platform

## Project Overview

IPL360 is a real-time cricket analytics platform built using Apache Kafka, Apache Spark, PostgreSQL, and Streamlit.

The platform processes real IPL ball-by-ball datasets to generate live analytics, player intelligence, team insights, predictive models, fantasy recommendations, and executive dashboards.
